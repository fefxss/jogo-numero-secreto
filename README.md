# 🎮 Jogo do Número Secreto

## 👀 Sobre
Projeto prático desenvolvido para aplicar conceitos de Lógica de Programação. Consiste em um jogo interativo onde o usuário precisa adivinhar um número secreto gerado aleatoriamente (por padrão, entre 1 e 10, podendo ser alterado no código).

Para deixar a experiência mais imersiva e acessível, foi integrado o **ResponsiveVoice**, um script de voz que narra os textos e dá dicas em áudio durante a partida.

▶️ **[Clique aqui para acessar e jogar!](https://jogo-numero-secreto-silk-nu.vercel.app/)**
